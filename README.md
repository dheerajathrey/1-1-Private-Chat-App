# 1-1 Private chat app

## Tools Used
-----------
* node.js
* socket.io
* javascript
* html, css

## Modules used
------------
* express
* socket.io

## Instructions to run the code
------------------------------
```
cd chatapp
npm install
node index.js
```

Go to localhost:3000 in the browser and login and each such user they can talk to other users privately.
